

**구현된 기능:**

**3D 시뮬레이션**
- 8×5 레일 그리드 (Three.js) — 천장 X·Y축 레일 시스템
- 28개 전기차 배치, 마우스 드래그/스크롤로 시점 조작

**알고리즘 엔진**
- **A\*** — 휴리스틱 기반 최적 경로 (Manhattan distance)
- **Dijkstra** — 완전 탐색 (최적 보장, 노드 더 많이 탐색)
- **Greedy BFS** — 고속 탐색 (빠르지만 최적 미보장)
- **다중 목표 정렬** — Nearest-Neighbor Greedy로 서비스 순서 최적화

**시각화**
- 🟡 노란 노드 = A* 열린 집합, 🟠 주황 = 닫힌 집합, 🟢 초록 = 최적 경로
- 충전 중 케이블 하강 애니메이션, 로봇 pulse glow
- 알고리즘 3종 탐색 노드 수 실시간 비교 바

**다음 단계**로 추가할 수 있는 것들:
- 실제 VLA 모델 연동 포트 인식 시뮬레이션
- 다중 로봇 동시 운영 스케줄링
- 충전 완료 시간 예측 모델
